Enhancing ML Model through Dynamic Feature Selection
(Automated Feature Selection System for Healthcare Data Prediction)

Project Overview

This group project focused on developing an automated feature selection system to optimize machine learning models for healthcare data predictions. By implementing advanced feature selection algorithms and dimensionality reduction techniques, the project improved model efficiency, reduced computational costs, and enhanced key performance metrics such as accuracy, F1-score, and AUC-ROC. The system is especially tailored for healthcare datasets, which are often high-dimensional and noisy.

Key Objectives

  • Automated Feature Selection
    Develop a dynamic feature selection framework to identify the most relevant features for healthcare data.
  • Dimensionality Reduction
    Apply algorithms like PCA to reduce the feature space while maintaining data integrity and minimizing information loss.
  • Model Performance Improvement
    Enhance key metrics such as accuracy, F1-score, and AUC-ROC across multiple ML models.

Technical Approach

Dataset and Preprocessing
Data Sources: Worked with high-dimensional healthcare datasets containing demographic, diagnostic, and treatment variables.
Preprocessing: Performed data cleaning, normalization, and encoding to ensure the dataset was consistent and ready for feature selection.

Feature Selection Techniques
Optimal Subset Feature Selection (OSFS): Used to identify minimal subsets of features with maximum predictive power.
Dynamic Optimal Subset Feature Selection (DOSFS): Dynamically updated the selected features as new data was added.
Principal Component Analysis (PCA): Reduced feature space by ~60%, eliminating redundancy and improving computational efficiency.

Model Development
Integrated feature selection with machine learning algorithms, including:

  • Support Vector Machines (SVM)
  • Logistic Regression
  • Decision Trees

Used cross-validation to validate model performance across multiple folds.

Performance Metrics
Accuracy: Improved by ~18%.
F1-Score: Boosted by ~22%, ensuring a balance between precision and recall.
AUC-ROC: Increased by ~15%, highlighting improved classification performance.

System Automation
Automated feature selection pipeline to dynamically adapt to evolving datasets without manual intervention.

Project Highlights

  • Significant Model Efficiency Gains: Reduced feature space by ~60%, decreasing training time while preserving data integrity.
  • Dynamic Feature Selection: Enabled the system to adapt to new data inputs, ensuring robust model predictions over time.
  • Enhanced Model Performance: Improved critical performance metrics across multiple models, demonstrating the versatility of the approach.
  • Scalable Solution: Designed the system to handle various datasets and integrate seamlessly with different ML pipelines.

Challenges Faced & How They Were Addressed

Handling High Dimensionality
Addressed computational challenges by integrating PCA and DOSFS for efficient dimensionality reduction.

Balancing Feature Reduction and Model Performance
Conducted rigorous testing to ensure that reducing features did not compromise predictive accuracy.

Algorithm Integration
Combined OSFS/DOSFS with existing ML models, optimizing compatibility and performance.

Impact and Outcomes

  • Improved Predictive Accuracy: Achieved significant performance gains across multiple machine learning models.
  • Streamlined Feature Selection: Reduced manual intervention through automation, increasing system scalability and adaptability.
  • Optimized Resource Usage: Lowered computational costs by decreasing the size of the feature space while maintaining model robustness.

Technologies & Tools

  • Programming Languages: Python (Pandas, NumPy, Scikit-learn)
  • Algorithms: OSFS, DOSFS, PCA
  • Machine Learning Models: SVM, Logistic Regression, Decision Trees
  • Visualization: Matplotlib, Seaborn (for feature importance & performance analysis)
  • Collaboration Tools: Jupyter Notebook, Git (version control)

Conclusion

This project demonstrates expertise in feature selection, dimensionality reduction, and optimizing machine learning workflows for high-dimensional healthcare datasets. By automating feature selection and improving model performance, the system provides a scalable and efficient solution for predictive analytics in healthcare, with applications in diagnostics, treatment planning, and resource allocation.
